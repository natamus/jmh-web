Mob Head Resources

On this page you can download textures to allow for modded entities to drop their head on death. You need the main mod from the CurseForge project page which can load the extra textures downloaded here.

Select the mods you want additional texture data for on the left side, and click on the right arrow to move them to download list. Click the download button to receive the JSON file with the combined data.

Move the downloaded JSON file (starting with jmh-textures) to the location %root%/data/jmh/jmh-textures-date.json.

(Minecraft %root% is where the 'saves'-folder is located)
